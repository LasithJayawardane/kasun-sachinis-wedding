# Starry Night Wedding Invitation 💍

A luxury, mobile-first digital wedding invitation for **Sachini & Kasun**.

## ✨ Features
- Twinkling animated starfield (Canvas)
- Shooting star effects
- Live countdown timer to July 31, 2026
- Glassmorphism card design
- WhatsApp RSVP with pre-filled messages + confetti
- Wishes & Blessings wall (localStorage)
- Background piano music (Clair de Lune)
- Fully responsive & mobile-optimized

## 🚀 Deploy

### Vercel (Recommended)
1. Push this folder to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Done — no configuration needed!

### Netlify
1. Drag & drop this entire folder onto [app.netlify.com/drop](https://app.netlify.com/drop)
2. Your site is live instantly!

## 📁 Files
- `index.html` — Structure & markup
- `style.css` — Luxury starry night styles
- `script.js` — Canvas, countdown, music, RSVP, wishes logic
- `vercel.json` — Vercel deployment config

Once deployed, share the URL on WhatsApp. The invitation works as a full-screen mobile experience.
